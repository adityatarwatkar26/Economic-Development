## Overview
Cleaned dataset of economic indicators (1995–2023) for countries including France, Brazil, Japan, Germany, Australia, China, UK, USA, India, and Canada.  
File: `Economic_Development_Cleaned_Dataset.xlsx`

## Columns (14)
- Country, Region, Income_Group, Development_Status  
- Year, Economic_Sector, Policy_Environment  
- GDP_USD_Billion, GDP_Growth_Rate_%, Inflation_Rate_%  
- Unemployment_Rate_%, Poverty_Rate_%, FDI_Inflow_USD_Billion  
- Human_Development_Index (HDI)

## Quick Insights
- Avg GDP highest: Canada (~$12.9T), China (~$12.9T)  
- HDI similar across income groups (~0.69–0.71)  
- Avg poverty slightly higher in Developed (23.8%) vs Developing (23.3%)  
- GDP growth ~3.5%, inflation ~7.5%, unemployment ~11% (averages)

## Use Cases
- Economic trend analysis  
- ML prediction (HDI, GDP growth)  
- Policy impact studies  
- Teaching / data visualization
